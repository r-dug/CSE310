Discrete Mathematics - Rings

Lecture Notes on Rings

Introduction to Abstract Mathematics

  • Abstract mathematics focuses on operations within a set.
  • A set exhibits closure if, when you perform an operation (addition, subtraction, multiplication, division) on any two elements within the set, the result is also an element of the set.
  • To show a set is not closed, you only need one counter example.
  • Example: Real numbers are not closed under division because you cannot divide by zero.
    • Real Numbers / Division: NOT closed
  • To demonstrate closure, an abstract construction that represents any arbitrary values is employed.

Properties of Sets in Mathematics

  • A group has more properties added to sets that allow for powerful concepts in mathematics, specifically for the operations of addition and multiplication.
  • Definition of a Group (G):
    • An operation must be defined within the group.
    • Sets have closure under the operation.
    • Groups have an additive inverse for each element.
    • Groups have an additive identity element.
    • Example: If G is a group under addition, then for any element a in G, the additive inverse is -a and the additive identity is 0.
  • Inverses are unique to each element within the group and come in pairs.
  • If any property fails, the group fails to be considered as such.

Defining Addition and Multiplication in Groups

  • Operations such as addition and multiplication can be abstractly defined and do not necessarily relate to standard arithmetic as we know it.
  • For instance, in computing, binary addition follows the exclusive or (XOR) operation, while binary multiplication follows logical AND operations.
  • Identity and Inverse in Groups:
    • Additive identity: Adding zero to any number returns the original number.
    • Additive inverse: Adding a number to its inverse (negative counterpart) returns the additive identity (zero).
    • Example for Real Numbers under addition: 3 + (-3) = 0
  • Real numbers are a group under addition but not under multiplication due to the element zero lacking a multiplicative inverse.

Identification of Groups Using Real Numbers

  • Real numbers under addition is a group.
  • Natural numbers under addition does not form a group due to lack of identity.
  • Integers under addition do form a group.
  • THIS WILL BE ON THE EXAM: Be able to demonstrate whether a set with a given operation is a group or not.

Exclusive Or and Logical AND Operations

  • Considering the set {true, false} which can also be represented as {0, 1},
    • Addition defined as XOR forms a group.
    • Logical AND operation does not form a group because zero does not have an inverse.

The Concept of Rings

  • A ring needs both addition and multiplication.
    • Set R is a group under addition.
    • Set R has an identity under multiplication.
    • Multiplication must distribute over addition.
  • A well-known ring is GF(2), also known as Galois field, which corresponds to binary arithmetic used in computing.
  • Upcoming Lectures: The next lecture will aim to prove that GF(2) is a ring.
